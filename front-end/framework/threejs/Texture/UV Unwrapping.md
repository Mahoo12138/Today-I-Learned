## 概述

UV Unwrapping 就是把 3D 模型的表面展开成 2D 的过程，用来定义每个顶点对应贴图上的哪个位置。

## 为什么叫 "UV"？

在 3D 图形中：

- **XYZ**：表示三维空间坐标
- **UV**：表示二维贴图坐标（贴图是图片，是 2D 的嘛）

每个顶点除了有 `position: (x, y, z)`，也会有 `uv: (u, v)`，告诉渲染器：
 “这个顶点贴在图片的哪个位置”。

## 重要性

| 原因                             | 解释                                          |
| -------------------------------- | --------------------------------------------- |
| 控制贴图如何覆盖模型             | 没有 UV，你的图片不知道怎么贴到模型上         |
| 避免贴图拉伸/错位                | 差的 UV 会让贴图变形，产生“贴花粘错了”的感觉  |
| 为烘焙贴图、法线贴图、AO贴图服务 | 这些都依赖 UV Map 的准确性                    |
| 自定义材质分布                   | 你可以在贴图上“画图”，UV 控制哪里显示什么内容 |
